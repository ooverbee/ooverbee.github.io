Digital Portfolio

Nanostring Microarray Analysis

In this R Markdown document, a Nanostring miRNA microarray experiment has been pre-processed and displayed for more detailed browsing. Differential expression data for each sample can be viewed with various interactive plots such as Heatmaply, ggVenn, and ggPlotly, as well as a small R Shiny app for searching for specific miRNA probes.

Geo-Omnibus Browser

Below is an R shiny app for accessing sequencing and array data from the NCBI GEO Omnibus database. By inputing experiment series, the user can choose specific samples from the series, browse for a target of interest, and apply the desired data transformation. Finally, the app generates an interactive plot as well as a data table with the values for the user to download.
